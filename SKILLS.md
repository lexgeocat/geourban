# Skills instalados en opencode (espanol)

Documento generado a partir de los skills instalados a nivel global en `~/.config/opencode/skills/` (5 repos: Graphify, UI UX Pro Max, ECC, Awesome Design.md y Get Shit Done). Las descripciones fueron traducidas al espanol; el nombre del skill se mantiene en ingles porque es el identificador que usa opencode.

## Como funcionan

- **Automatico**: opencode carga el skill solo cuando su descripcion coincide con lo que pides.
- **Manual**: invocalo con `/skill <nombre>` (o el comando de tu version de opencode).
- **Importante**: opencode no recarga la config en caliente. Tras instalar/modificar skills, **salite y volve a abrir opencode**.
